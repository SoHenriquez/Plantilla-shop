# Api
1. Clonar proyecto
2. ```npm install```
3. Clonar archivo ```.env.template``` y renombrarlo ```.env```
4. Cambiar las variables de entorno
5. Levantar la DB
```
docker-compose up -d
```
6. npm run start:dev
